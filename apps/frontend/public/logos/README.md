# Logos de fundaciones (Mis Fundaciones - Dashboard)

Cada archivo debe llamarse según el **slug** del micrositio Bonda de la organización, con extensión `.jpg`.

Ejemplos:
- `club-plato-lleno.jpg` → Plato Lleno
- `beneficios-fundacion-padres.jpg` → Fundación Padres
- `beneficios-biblioteca-rurales.jpg` → Biblioteca Rurales Argentinas
- `beneficios-haciendo-camino.jpg` → Haciendo Camino

Si no existe el archivo para un slug, en el dashboard se muestran las iniciales del nombre de la fundación.
